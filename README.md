`npm start`

## How to run test
### To open the UI window and click on the test you want to run in browser.\
`npx cypress open` or\
`npm run cypress`

### To run test just in console as headless browser.
`npx cypress run` or\
`npm test`\
With `run` options, we will have generated videos as evidences in <cypress\videos>
